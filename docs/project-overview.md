# Project Overview

## Problem

Many organisations understand the importance of cybersecurity, but struggle to translate concepts like Zero Trust into practical, actionable steps. 

In most cases, businesses either:
- Do not know where to start
- Implement partial solutions without a clear strategy
- Invest in tools without understanding their real security gaps

This creates fragmented security environments that remain vulnerable to modern threats such as phishing, credential theft, and lateral movement.

---

## Solution

Zero Trust Adapt is designed to bridge the gap between cybersecurity theory and real-world implementation.

The project introduces a structured assessment approach based on a survey covering key security domains:

- Users
- Endpoints & Devices
- Internet Traffic
- Networks
- Applications
- Data Loss & Logging
- Automation & Steady State

Based on the combination of responses, the system generates a tailored report that:

- Identifies current security gaps
- Highlights what is already in place and can be improved
- Recommends what needs to be implemented
- Provides clear, step-by-step actions
- Suggests relevant software and hardware solutions

---

## Objective

The main objective of this project is to simplify the adoption of Zero Trust principles by making them accessible, structured, and aligned with business reality.

Instead of generic advice, organisations receive guidance adapted to:

- Their size
- Their current level of maturity
- Their existing security controls

---

## Value

This approach allows organisations to move from uncertainty to clarity, providing a realistic and prioritised path towards a stronger security posture.

The project aims to demonstrate how cybersecurity decisions can be structured, contextual, and directly aligned with real operational needs.
